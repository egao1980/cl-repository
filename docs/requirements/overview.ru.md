# Требования CL Repository (RU)

Исходный документ: `docs/requirements/overview.md`

## Проблема

В Common Lisp нет стандартного эквивалента PyPI/crates.io/npm с полноценной поддержкой нативных зависимостей.

## Цели

1. OCI-совместимая дистрибуция.
2. Мультиплатформенные native overlays.
3. ASDF-native установка.
4. Минимум тяжелых зависимостей.
5. Обратная совместимость для pure-Lisp пакетов.
6. Bootstrap экосистемы через экспорт из Quicklisp/Ultralisp.

## Не-цели

- Реализация собственного registry-сервера.
- Распространение FASL бинарников.
- Подпись пакетов на уровне приложения (делегируется OCI-инфраструктуре).
